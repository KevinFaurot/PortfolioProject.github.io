COVID Data Exploration in SQL

In this project I looked at COVID-19 data and compared total cases with total deaths and found a percentage of deaths per country. I also looked at infection rates as well as death rates per population broken down by continent. Later in the exploration I created a CTE to create a rolling total of people vaccinated as a percent of the population and also created a view for later data visualizations.

Nashville Housing Data Cleaning in SQL

For this project the data was cleaned in SQL by standardizing the date format, determine what to do with null values, making the data more usable separating the property addresses into street, city, and state. Finally, the data was cleaned further by changing the Sold as Vacant field to make it more readable and then duplicates and unused columns were removed. While it is not common practice to delete data, it was done for demonstration purposes.
